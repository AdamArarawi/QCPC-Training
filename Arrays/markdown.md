# Python Arrays Tutorial

In Python, arrays can be handled using **lists** or the `array` module. We'll focus on **lists**, which are more commonly used.

---

## 1. Creating Arrays (Lists)

```python
# Using a list to store numbers
numbers = [1, 2, 3, 4, 5]
print("Numbers:", numbers)

# Using a list to store strings
fruits = ["apple", "banana", "cherry"]
print("Fruits:", fruits)
